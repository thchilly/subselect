# papers/

Project literature kept alongside the design docs for offline reference. PDFs are intentionally tracked in git (single-MB scale, immutable, useful when a Claude Code session needs to consult a paper without web access). Add new entries with one-line citations below.

When you add a paper here, drop a line in this index — agents skim it to know what's intentional and what would be cruft.
