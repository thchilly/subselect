# Historical paper outputs

These xlsx files are the published-paper outputs (Tsilimigkras et al., April
2026 *climate-model subselection over Greece*). They are retained for
historical reference and to prove reproducibility at the `phase-0-complete`
git tag (commit `fb7527f`).

**They are not a regression target for the current package.** The methodology
has since been corrected in the M-CORRECT step (`methodology-corrected` tag):

1. **Cos(latitude)-weighted spatial mean for σ_obs.** Paper-era code used
   an unweighted spatial mean (`.mean(dim=['lat', 'lon'])`) that was
   inconsistent with the cos-weighted means everywhere else in the pipeline.
2. **Native 0.5° W5E5 reference for σ_obs.** Paper-era code used the
   cmip6-grid upscaled product (1.25° × 1.875°) for σ_obs. The corrected
   pipeline reads from the native 0.5° single-grid product at
   `Data/reference/monthly_05/` (loader: `subselect.io.load_native_w5e5`).

Both corrections shift TSS values slightly (typically <1 % per model), and
therefore shift HMperf and the model rankings. The corrected numbers are
canonical for the current package; these xlsx files capture the paper-era
state for traceability.

## Files

| File | What it is |
|---|---|
| `assess_cmip6_composite_HMperf_greece.xlsx` | Composite HMperf (`_HMperf` per period) for the 35 canonical CMIP6 models over Greece. |
| `assess_cmip6_<var>_mon_perf_metrics_all_seasons_greece.xlsx` | Per-variable {std_dev, corr, bias, rmse, crmse, bias_score, tss, tss_hirota} per period for the 35 models. One file per variable in `{tas, pr, psl, tasmax}`. |
| `assess_long_term_change_spread_greece.xlsx` | End-of-century (2081–2100) − pre-industrial (1850–1899) change signals per (model, variable, period) under SSP5-8.5. |
| `assess_long_term_spread_greece.xlsx` | End-of-century absolute spatial-mean climatology per (model, variable, period) under SSP5-8.5. |
| `models35.txt` | Canonical 35-model ordering (1..35) used by every figure marker. |

## How these were produced

Generated by the legacy paper-era notebooks under `legacy/cmip6-greece/`:

- `assess_cmip6_*_perf_metrics_*.xlsx` and `assess_cmip6_composite_HMperf_*.xlsx` from `GR_model_performance_HM.ipynb` (cells building `perf_metrics` and `ranked_full`).
- `assess_long_term_*_spread_*.xlsx` from `GR_model_spread.ipynb` (cell 4).

Cross-reference: `documentation/methods.tex § Methodology corrections
(post-paper)` for the rationale; `docs/refactor.md § Phase 0 closeout` for
the timeline.
